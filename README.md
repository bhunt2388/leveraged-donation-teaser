# Leveraged Donation Strategy — Develop My Credit 1 LLC

Interactive teaser deck + email template for the 5:1 charitable deduction tax strategy.

## Live URLs

- **Presentation:** https://leveraged-donation-teaser.vercel.app
- **Email template:** https://leveraged-donation-teaser.vercel.app/email.html

## Files

| File | Purpose |
|------|---------|
| `index.html` | Full interactive teaser (10 slides, live calculator, nav dots) |
| `email.html` | HTML email template (table-based, inbox-safe) |

## Workflow — Edit & Deploy

Every push to `main` auto-deploys to production via Vercel. No manual deploy step needed.

```bash
# 1. Make changes to index.html or email.html
# 2. Commit and push
git add .
git commit -m "describe your change"
git push origin main
# → Vercel picks it up automatically, live in ~15s
```
